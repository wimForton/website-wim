Particle System

This is an exercise in Typescript, Angular and ng Material.

The endgoal is to have a complete framework for creating 3D content and animation.

How to use it:

Add operators and tweak parameters.

At the moment the layout works best as a desktop application since there are many features and parameters that have to be displayed at once.
